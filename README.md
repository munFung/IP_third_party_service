# Setup
1. Download latest SQLite
````
https://sqlitebrowser.org/blog/version-3-13-0-released/
````
2. install dependencies
````
pip install -r requirements.txt
````
3. Use SQLite click *Open Database* > *[project_name]* > *instance* > *example.db*
